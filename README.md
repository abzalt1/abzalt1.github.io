# Model Portfolio & Business Solutions

A high-performance, visually stunning portfolio and admin panel built for fashion models and business professionals.

## 🚀 Tech Stack

### Frontend & Core
*   **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
*   **Library**: [React 19](https://react.dev/)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/) for precision UI
*   **Icons**: [Remix Icon](https://remixicon.com/)

### Infrastructure & Storage
*   **Hosting**: [Vercel](https://vercel.com/) (Edge-ready deployment)
*   **Media Storage**: [Cloudflare R2](https://www.cloudflare.com/products/r2/) (S3-compatible, zero egress fees)
*   **Optimization**: Custom image optimization pipeline for ultra-fast loading

## 🎨 Key Features
*   **Minimalist Design**: Clean, typography-focused aesthetic.
*   **Dark Mode**: Native support with system preference detection.
*   **Dynamic Gallery**: Managed via a custom admin panel integrated with Cloudflare R2.
*   **Responsive**: Fully optimized for mobile and desktop viewing.

## 🛠 Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
